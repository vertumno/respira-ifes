# Respira Cefor — Apresentação

Apresentação institucional do projeto **Respira**, do Núcleo de Qualidade de Vida no Trabalho (QVT) do Cefor/Ifes. Criada para exibição em datashow durante evento presencial no Campus Vitória.

## Como usar

Abra [`apresentacao-respira.html`](apresentacao-respira.html) no navegador e pressione `F11` para tela cheia.

**Navegação:**
- `→` / `←` ou `Espaço` — avançar / voltar entre slides
- Setas laterais na tela, indicadores na base ou swipe (touch)

A apresentação é 100% local — funciona offline (exceto pelas fontes Google, opcionais).

## Slides

1. Hero — "Respire. De verdade."
2. O que é o Respira
3. Benefícios comprovados (com referências científicas)
4. Nossa história — origem presencial em 2022 (período de pandemia)
5. Fotos do presencial (2022)
6. Pratique agora — sozinho no site ou ao vivo toda semana
7. Foto da sessão online (2026)
8. QR code do grupo de WhatsApp
